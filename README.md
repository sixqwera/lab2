# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их
двоичных значений.

### Алгоритм решения

Начинаем с момента создания списка, после создаем доп. список(там будут складироваться уже переведенные цифры) с теми же элементами и прогоняем весь этот список через функцию toBin где получаем на выходе двоичные числа, а позже их и выводим.
