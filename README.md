# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их
двоичных значений.

### Алгоритм решения

getlist(): Рекурсивная функция ввода. Считывает строку, проверяет её на пустоту и с помощью Char.IsDigit фильтрует ввод, пропуская только числа.
toBin(n): Вызывает стандартный метод .NET для преобразования десятичного целого числа в его строковое двоичное представление.
main:
Заполняет список numbers через вызов getlist.
С помощью List.map применяет функцию toBin к каждому элементу списка, создавая новый список строк after_toBin.
Выводит оба списка в консоль.

# Задание 2

## Задача 1

### Текст задачи

Список содержит десятичные цифры. Получить строку, перечисляющую эти
цифры. Например, для списка [1; 3; 1; 7] получаем строку «один; три; один;
семь».

### Алгоритм решения

Сбор данных (getlist): Рекурсивно читает ввод. Если введена ровно одна цифра, она добавляется в список. Если введена буква или пустая строка — ввод прекращается.
Перевод (rus_words): С помощью конструкции match (сопоставление с образцом) заменяет каждое число от 0 до 9 на соответствующее слово («один», «два» и т.д.).
Преобразование (List.map): Проходит по всему списку чисел и превращает каждое из них в слово, создавая новый список слов.
Сборка строки (List.fold): Схлопывает (сворачивает) список слов в одну длинную строку.
На каждом шаге к накопленному тексту all прибавляется пробел и новое слово.
