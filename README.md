# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их
двоичных значений.

### Алгоритм решения

getlist(): Рекурсивная функция ввода. Считывает строку, проверяет её на пустоту и с помощью Char.IsDigit фильтрует ввод, пропуская только числа.
toBin(n): Вызывает стандартный метод .NET для преобразования десятичного целого числа в его строковое двоичное представление.
main:
Заполняет список numbers через вызов getlist.
С помощью List.map применяет функцию toBin к каждому элементу списка, создавая новый список строк after_toBin.
Выводит оба списка в консоль.
