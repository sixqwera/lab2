# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего цифры от 1 до 9, получить список из их
двоичных значений.

### Алгоритм решения

getlist(): Рекурсивная функция ввода. Считывает строку, проверяет её на пустоту и с помощью Char.IsDigit фильтрует ввод, пропуская только числа.
toBin(n): Вызывает стандартный метод .NET для преобразования десятичного целого числа в его строковое двоичное представление.
main:
Заполняет список numbers через вызов getlist.
С помощью List.map применяет функцию toBin к каждому элементу списка, создавая новый список строк after_toBin.
Выводит оба списка в консоль.

# Задание 2

## Задача 1

### Текст задачи

Список содержит десятичные цифры. Получить строку, перечисляющую эти
цифры. Например, для списка [1; 3; 1; 7] получаем строку «один; три; один;
семь».

### Алгоритм решения

Ввод данных: После запуска программа просит вас вводить цифры по одной. Вы печатаете символ и нажимаете Enter.
Проверка: Если вы ввели одну цифру (от 0 до 9), программа запоминает её и ждёт следующую. Если вы ввели что-то другое (букву или несколько знаков), программа выведет «Ошибка!» и предложит попробовать снова.
Завершение: Чтобы закончить ввод, нужно просто нажать Enter на пустой строке.
Обработка: Программа собирает все введённые цифры в список, а затем с помощью функции fold «склеивает» их названия на русском языке в одну длинную строку.
Результат: В конце вы видите сам список цифр и итоговую строку слов (например: «один пять три»).
